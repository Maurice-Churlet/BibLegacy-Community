# Presentation Produit

## BibLegacy en une phrase
BibLegacy automatise le tri et le renommage de photos de courses a dossard a partir de la lecture OCR des numeros visibles sur les images.

## Probleme resolu
Apres un evenement, les photos doivent etre retrouvees rapidement par numero de dossard. Sans outil adapte, cela implique un tri manuel long, repetitif et couteux.

## Reponse apportee
BibLegacy propose un poste de travail unique pour :
- preparer un lot de photos
- appliquer un profil OCR adapte
- lancer une production en masse
- corriger les cas difficiles via une plateforme web locale
- produire une page HTML et un index photo reutilisables pour afficher les images d'un dossard sur une page WordPress
- conserver des rapports et traces d'exploitation

## Positionnement actuel
Le produit courant est centre sur :
- RapidOCR comme moteur principal
- des profils de traitement canoniques
- des mecanismes de renfort comme le fallback et TrOCR selon le besoin
- une exploitation locale, orientee operateur

## Ce que voit un utilisateur
- une interface desktop en onglets
- un lancement de production simple
- une progression visible
- des dossiers de sortie directement exploitables
- une solution de rattrapage pour les cas non reconnus
- une sortie web exploitable pour consultation locale ou integration WordPress

## Pour quels contextes
- associations sportives
- organisateurs de trails, courses sur route, cyclos ou triathlons
- photographes evenementiels
- prestataires de traitement photo a dossards

## Idee force
Le produit ne promet pas une lecture parfaite sur chaque image. Il reduit fortement la charge manuelle, structure le traitement et donne a l'operateur un flux de travail complet de l'entree jusqu'a la correction finale.
