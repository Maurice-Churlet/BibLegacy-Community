# Documentation Marketing

Ce dossier contiendra la documentation de presentation du produit pour des associations sportives, organisateurs, photographes et prestataires de manifestations a dossard.

## Objectifs
- presenter clairement le probleme resolu
- expliquer les benefices metier du produit
- fournir un support de demonstration et de discussion commerciale

## Fichiers prevus
- `presentation-produit.md`
- `benefices-metier.md`
- `faq-commerciale.md`
- `parcours-evenement.md`

## Fichiers disponibles
- `presentation-produit.md` : discours de presentation general du produit
- `benefices-metier.md` : gains operatoires et valeur d'usage
- `faq-commerciale.md` : reponses courtes aux objections et questions frequentes
- `parcours-evenement.md` : narration simple de l'usage avant, pendant et apres une course

## Usage recommande
Ce dossier sert a preparer :
- une demonstration produit
- un support de discussion avec une association ou un organisateur
- une base de contenu pour futures plaquettes, pages web ou presentations

## Regle
La documentation marketing doit deriver de la documentation produit validee. Elle simplifie le message, mais ne doit pas inventer des capacites non verifiees.
