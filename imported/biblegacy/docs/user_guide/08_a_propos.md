# 08 - A propos

## Produit
**BibLegacy** est l'interface graphique principale du projet de traitement OCR de dossards present dans ce depot.

L'application combine :

- une interface desktop en Python/Tkinter,
- un pipeline OCR oriente production,
- des fonctions de calibrage, de comparatif et d'audit,
- une generation de rapports et de traces pour le suivi des executions.

## Technologies principales
Le produit s'appuie notamment sur :

- **Python** pour l'application,
- **Tkinter** pour l'interface,
- **RapidOCR** comme moteur OCR principal de l'application,
- **TrOCR** comme renfort optionnel sur les cas plus incertains.

## Positionnement actuel
L'etat actuel du produit privilegie :

- la robustesse d'exploitation,
- la lisibilite des profils,
- la tracabilite des executions,
- la capacite a comparer plusieurs strategies de lecture sur un meme lot.
