# 🤖 Moteur OCR

Le moteur OCR (Optical Character Recognition) est le "cerveau" du programme. Il analyse l'image pour y trouver des caractères numériques.

- **RapidOCR** : moteur principal de l'application actuelle.
- **TrOCR** : renfort optionnel sur certains cas incertains, selon les options actives.

---
💡 *Astuce : Si un dossard n'est pas détecté, testez d'abord Fallback, puis TrOCR si la couverture prime sur la vitesse.*
