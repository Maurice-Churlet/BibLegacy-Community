# 📂 Dossier Projet

Le Dossier Projet est le centre névralgique de votre travail. En sélectionnant un dossier ici, le programme crée automatiquement la structure suivante :

- **Production/** : Mettez ici les photos de la course à traiter.
- **Calibrage/** : Contient les images servant à régler les moteurs.
  - *Express* / *Standard* / *Deep* : Sous-dossiers pour graduer vos tests.
- **Tests OCR/** : Historique et résultats.

---
✅ *Avantage : Plus besoin de renommer vos fichiers en T ou P !*
