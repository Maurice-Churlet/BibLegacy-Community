# Mode Production

Le **mode Production** est le moteur principal pour traiter vos photos de course en masse.

### Utilisation :
- Chargez vos photos dans le dossier **Entrée**.
- Configurez le dossier **Sortie**.
- Cliquez sur **LANCER LA PRODUCTION**.

### Automatisation :
Le programme utilise les meilleurs paramètres trouvés lors des phases de calibrage précédentes pour chaque moteur et chaque mode (Express, Standard, Deep).

### Résultats :
- Les numéros détectés sont écrits dans les fichiers (ex: via le renommage ou l'index JSON).
- Un rapport de performance est généré à la fin.
