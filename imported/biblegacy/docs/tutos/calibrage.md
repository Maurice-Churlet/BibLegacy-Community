# Calibrage des Moteurs OCR

Le **calibrage** est une étape essentielle pour obtenir la meilleure précision possible. 

### À quoi ça sert ?
Il permet de tester automatiquement des dizaines de combinaisons de paramètres (echelle, contraste, seuils moteur) sur un echantillon d'images de test.

### Fonctionnement :
1. Verifiez le profil et les reglages que vous voulez optimiser.
2. Cliquez sur **Démarrer le Calibrage**.
3. Le programme cherche les réglages qui maximisent le score de détection.
4. Les résultats sont sauvegardés et utilisés automatiquement en mode Production.

> [!TIP]
> Utilisez des images représentatives de votre course (même luminosité, même type de dossards).
