# Exemples de Phrases pour Copilot

Voici des exemples simples de phrases que vous pouvez ecrire dans le chat Copilot.

## Pour savoir quoi faire

- `je ne sais pas quel test lancer`
- `je veux verifier si tout marche encore`
- `je veux savoir par ou commencer`

Commande conseillee : `/BibLegacy Start Here`

## Pour la GUI

- `j'ai change un bouton et je veux verifier la GUI`
- `un onglet ne repond plus correctement`
- `je veux verifier que l'interface n'a pas casse`

Commande conseillee : `/BibLegacy GUI Regression`

## Pour la config et les profils

- `j'ai change la sauvegarde des profils`
- `je veux verifier la compatibilite de la config`
- `je veux verifier les chemins runtime`

Commande conseillee : `/BibLegacy Persistence Migration`

## Pour la production OCR

- `j'ai change le pipeline OCR et je veux verifier le run`
- `je veux verifier le fallback`
- `je veux verifier les sorties de production`

Commande conseillee : `/BibLegacy Production Pipeline`

## Pour la release

- `je veux verifier si la release est prete`
- `je veux preparer la livraison Windows`
- `je veux savoir quoi controler avant d'envoyer`

Commandes conseillees :
- `/BibLegacy Release Readiness`
- `/BibLegacy Windows Release`

## Pour la documentation

- `j'ai modifie la documentation`
- `je veux reconstruire le site de doc`
- `je veux preparer la synchronisation communautaire`

Commande conseillee : `/BibLegacy Docs Sync`

Taches utiles :
- `Build Public Docs`
- `Serve Public Docs Locally`
- `Sync Community Docs`
