# Questions Frequentes Copilot

## Je ne sais pas quoi demander a Copilot

Commencez par `/BibLegacy Start Here` et ecrivez une phrase simple sur votre besoin.

Exemple : `je veux verifier que la GUI marche encore`

## Je ne sais pas quelle tache lancer

Si vous voulez verifier plusieurs zones d'un coup, lancez `Quick Validation Pack`.

Si vous voulez seulement ouvrir l'application, lancez `Start BibLegacy GUI`.

## Je ne comprends pas la difference entre les commandes et les taches

- Une commande Copilot sert a vous guider, expliquer, choisir ou analyser.
- Une tache VS Code sert a lancer une action concrete comme un test, un build ou la GUI.

## Je veux la solution la plus simple possible

Retenez seulement ceci :

1. pour savoir quoi faire : `/BibLegacy Start Here`
2. pour verifier vite : `Quick Validation Pack`
3. pour ouvrir l'application : `Start BibLegacy GUI`

## Je veux voir la documentation dans mon navigateur

Lancez `Serve Public Docs Locally`, puis ouvrez `http://127.0.0.1:8000`.

## Je viens de modifier l'interface

Utilisez `/BibLegacy GUI Regression`, puis lancez `GUI Regression Suite`.

## Je viens de modifier la config ou les profils

Utilisez `/BibLegacy Persistence Migration`, puis lancez `Persistence Regression Suite`.

## Je viens de modifier la production OCR

Utilisez `/BibLegacy Production Pipeline`, puis lancez `Production Smoke Suite`.

## Je veux preparer une livraison

Utilisez `/BibLegacy Release Readiness`.

Si vous devez produire l'executable Windows, lancez ensuite `Build Windows EXE`.
