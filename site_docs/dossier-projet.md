# Dossier Projet

Le **Dossier Projet** est la racine de travail de BibLegacy.

## Structure creee automatiquement

- `Production`
- `Calibrage`
- `Test_OCR`
- `Tests OCR`
- `Base photos`
- `Output`

## Role des dossiers

- `Test_OCR` : comparatifs courts et longs
- `Calibrage` : images de reference pour les reglages et apercus
- `Base photos` : reservoir large d images reelles pour les preparatifs avances
- `Output` : sortie recommandee pour les traitements OCR et exports web
- `Production` : rangement optionnel de lots source
- `Tests OCR` : dossier historique ou secondaire pour certains usages de test

## Idee cle

Une fois le **Dossier Projet** choisi, l utilisateur n a pas besoin de reconstruire la logique des chemins : il lui suffit surtout de mettre ses fichiers dans les bons dossiers.
